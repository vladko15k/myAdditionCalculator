# myAdditionCalculator [v1.0]

Here is a concise explanation and the source code for a simple C# addition calculator, perfect for understanding the basics of input, output, and data conversion.

C# Simple Addition Calculator
This program demonstrates four fundamental concepts in C#:

Console.ReadLine(): Reading input from the user as a string.

int.Parse() / double.Parse(): Converting strings into numerical data types.

Variables: Storing values in memory.

String Interpolation: Formatting the output using the $ symbol.
